----------Pseudo Shell-----------
Name: Tyler Sellin
UO ID: tsellin
This project simulates some basic linux shell commands.
It has functionality for ls, pwd, mkdir, cd, cp, mv, rm, and cat
To run the project, download the zip file, and build it by
running the Makefile(run make in the terminal).
This will generate an executable titled pseudo-shell.
To launch an interactive shell, simply type ./pseudo-shell to run the program
without any flags. You will then be able to continuously type commands, just like the linux shell.
To exit, simply type quit. To run it in file mode, add the -f flag, and then the input file.
It takes in commands line by line, and commands can be chained on a single line by including a
semicolon in between. Output will go into a file titled output.txt
